### 🚀 Featured Projects

#### ✈️ SkyElite (3rd Place - National AI Hackathon)
A passport and budget-aware AI travel recommendation system built with **Next.js, TypeScript, FastAPI, and LangGraph**. Features a robust 7-node StateGraph pipeline to filter destinations and returns ranked packages with confidence levels. 
*Tech: LangGraph, FastAPI, Supabase, Groq API*

#### 🧠 Nexus AI - Multi-Agent Research Assistant
A stateful multi-agent system coordinating 4 specialized agents (Clarity, Research, Validator, Synthesis) for automated industry research. Uses Server-Sent Events (SSE) to stream real-time markdown reports token-by-token.
*Tech: LangGraph, FastAPI, Server-Sent Events, Tavily*

#### ⚙️ FastAPI Blog Engine
A highly scalable RESTful API built with **FastAPI and SQLAlchemy**, featuring modular routing, dependency injection, and industry-standard OAuth2 authentication with JWT and strict Pydantic data validation.
*Tech: Python, FastAPI, SQLAlchemy, JWT*

#### 🤖 n8n AI Workflow Automation Suite
An enterprise-grade automation suite integrating Google Gemini with **n8n** and Google Workspace APIs to process documents and power conversational AI workflows.
